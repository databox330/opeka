Animations

Каталог анимационных пресетов. Все анимации в проекте — через data-anim атрибут. GSAP / SplitType / Lenis в коде секций запрещены: только пресеты в src/animations/presets/. Если нужна новая анимация — обсуждение, потом новый пресет, потом запись здесь.

reveal-up

Появление из прозрачности с лёгким сдвигом снизу-вверх при попадании в зону видимости. Универсальный пресет для текстов, кнопок, карточек. Реализован на нативном IntersectionObserver + CSS-transition, без GSAP. Уважает prefers-reduced-motion.

Параметры: data-anim-delay="0.1" (сек), data-anim-duration="0.3" (сек, по умолчанию 0.3).

HTML 
<h2 data-anim="reveal-up">Появится снизу-вверх</h2>
<p data-anim="reveal-up" data-anim-delay="0.1">С задержкой 100мс</p>
